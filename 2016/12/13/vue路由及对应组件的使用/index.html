路由

1.安装

npm i vue-router -S

2.导入vue-router

import vueRouter from vue-router

3.导入需要设置路由规则的组件

import home from ‘./components/home/home.vue’

4.创建路由实例,设置规则

const router = new VueRouter({
routes: [
{ path: ‘/category’, component: category },
]
})

5.创建和挂载实例,通过router配置参数注入路由

new Vue({
el: “#app”,
router,
render: function(createElement) { //项目一启动之后，呈现给用户的第一个组件
return createElement(App)
}
})

vue-resource

概念

对ajax的封装,后台请求数据

使用

安装
npm i vue-source -S

导入
在main.js中import vueResource from ‘vue-resource’

集成
Vue,use(vueResource)

$route和$router的区别

$route是设置路径相关,比如监控路由,获取参数

$router执行者,执行一个动作
写js代码实现路由的前进后退和回退,也叫编程式导航

goBack(){
    this.$router.go(-1)
  }

导航分类

编程式导航:写js代码实现路由的前进后退和回退
声明式导航:事先声明在template标签里面

<router-link to="/news/newslist">
