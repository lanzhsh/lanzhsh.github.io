vue项目搭建

By LanZhiSheng

发表于 2016-12-23

手动搭建脚手架

1.配合使用的插件有

webpack-dev-server
html-webpack-plugin
两个插件的配合使用最后将入口文件main.js打包成bundle.js文件

1.创建文件及文件夹

src:项目的源代码,src里面有App.vue(根组件),
package.json:项目配置文件
webpack.develop.config:开发阶段的webpack配置文件
template.html:html-webpack-plugin会将该文件参照，在内存中生成一个指定名称的(index.html)的文件,也就相当于每次改变不用重新打包

2.开发阶段配置文件书写
var path = require('path')
//每次刷新不用重新打包
var HtmlWebpackPlugin = require
('html-webpack-plugin')
module.exports = {
//打包的入口文件
entry: './src/main.js', 
output: {
    //bundle.js放的目录
    path: __dirname, 
    //打包出来的文件名称
    filename: 'bundle.js' 
},
module: {
//loader预处理文件
loaders: [{
    test: /\.vue$/,
    loader: 'vue-loader'
},
{
    test: /\.css$/,
    loader: 'style-loader!css-loader'
},
{
    test: /\.(ttf|png)$/,
    loader: 'url-loader'
}
]
},
plugins: [
    //初始化实例
    new HtmlWebpackPlugin({
        filename: 'index.html', 
    //内存中生成的文件名称
        template: './template.html'
     //模版文件
    })
]
}

可以用vue-cli搭建

安装
npm i -g vue-cli
生成项目

vue init webpack-simple vue_simple_demo

切换到根目录安装依赖的包

npm i

补充

脚手架里面有个.babelrc文件,webpack集成了babelrc将ES6语法转为ES5
