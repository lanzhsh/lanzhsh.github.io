vue组件书写代码段

By LanZhiSheng

发表于 2016-12-15

VSCode编辑器

设置–用户–代码块–选择vue–复制以下代码

"my_vue": {
    "prefix": "my_vue",
    "body": [
        "<template>",
        "   <div class='tmpl'>",
        "        $0",
        "   </div>",
        "</template>",
        "    ",
        "<style scoped>",
        "    ",
        "</style>",
        "    ",
        "<script>",
        "    import common from '../
        ../common/common.js'",
        "    ",
        "    export default {",
        "        data() {",
        "            return {",
        "            }",
        "        },",
        "        created() {",
        "            ",
        "        },",
        "        methods: {",
        "            ",
        "        }",
        "    }",
        "</script>"
    ],
    "description": "快速创建vue模板"
}

几个主要属性

data:Vue实例的数据对象,组件的定义只接受函数
created:检测组件刚创建的时候执行的代码
methods:可以直接混入到vue实例中,可以通过this(也就是vue实例)访问这些方法
watch:检测一些变化,如路由值
