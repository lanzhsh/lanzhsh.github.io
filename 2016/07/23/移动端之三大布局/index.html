移动端之三大布局

By LanZhiSheng

发表于 2016-07-23

流式布局

1.概念

就是百分比布局;
大盒子宽度设置百分比,里面内容用rem作为单位;

2.rem原理

rem是一个单位,是用媒体查询修改html的font-size来改变rem值;

3.rem的应用

A.设计稿尺寸(一般为640px/750px)转化为rem值,分为若干份每一份大小是rem的单位;
B.把设计图中其他的尺寸转化为rem值;
C.在不同设备动态设置font-size值:

document.querySelector('html').style.fontSize
=window.screen.width/20+'px';

D.如果是在浏览器中测试,需设置,因为固定设备屏幕大小固定:

window.onresize=function(){
    querySelector('html').style.fontSize
    =window.screen.width/20+'px'}

弹性布局

1.特性
是css3出来的新特性,所以对IE来说兼容性不好;
2.用法
A.父元素设置的属性:

调整主轴位置:flex-direction:row/column/row-reverse/column-reverse;
主轴对齐方式:justify-content:flex-start/flex-end/center/space-around/space-between;
侧轴对齐方式:align-item:flex-start/flex-end/center/stretch;
换行:flex-wrap:nowrap/wrap;

B.设置子元素属性

flex:1//设置规定宽度,均分剩下宽
order:设置子元素的排列顺序
align-self:设置子元素侧轴位置

响应式布局

1.概念:

web内容可以随着设备尺寸大小而变化

2.原理:
A.媒体查询:

/ 小屏幕（平板，大于等于 768px） /
@media (min-width: @screen-sm-min) { … }
/ 中等屏幕（桌面显示器，大于等于 992px） /
@media (min-width: @screen-md-min) { … }
/ 大屏幕（大桌面显示器，大于等于 1200px） /
@media (min-width: @screen-lg-min) { … }
